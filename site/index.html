Architecture

Built for Performance

Nexus uses a modern architecture designed for speed, security, and extensibility. Every component runs locally on your machine.

  • Tauri 2.x for native performance with web technologies
  • TypeScript sidecar with JSON-RPC 2.0 over stdio
  • Smart routing between multiple LLM providers
  • Semantic + prompt caching to reduce costs
  • SQLite for local storage, zero cloud dependencies
  • Transparent tool calls with real-time execution logs
